Skill 起動経路 (top 20) LLM 自律 vs ユーザー手動 同一 skill に対する skill_tool (🤖 LLM が Skill ツールで自律発火) と user_slash_command (👤 ユーザーが /foo で手動発火) の件数比。Mode は dual (両方観測) / llm-only (Skill ツールのみ) / user-only (slash command のみ) の 3-way。dual の LLM 率は tool / (tool + slash)。低い (= ユーザー手動依存が強い) skill は description / trigger が弱く LLM が「思いつかない」候補。

Skill lifecycle (top 20) 初回 / 直近 / トレンド 各 skill の first_seen / last_seen + 直近 30 日件数 + 全期間件数 + トレンドラベル (📈 加速 / ➡️ 安定 / 📉 減速 / 🌱 新規)。first_seen から 14 日未満はトレンド判定不可で 🌱 新規。それ以外は recent_rate / overall_rate 比 (>1.5 加速、<0.5 減速、それ以外安定)。並び順は last_seen 降順。
